# 熱區影像人流辨識與預測系統：結合深度學習的即時監控與預測平台

## 1. 主題（專題方向）

**熱區影像人流辨識與預測系統**

開發一套結合影像辨識與時間序列預測模型的人流分析系統，可即時偵測場域中人數並預測未來一段時間的流量變化，用於支援校園空間管理與資源調度決策。

---

## 2. 動機（為什麼想做這個）

- **場域人流資訊掌握困難：** 商場、展覽會場等公共空間缺乏有效的人流監控與預測手段，常造成擁擠、動線混亂等問題。
- **紅外線或壓力感測器精確度有限：** 雖然紅外線或壓力感測器可用來估算人數，但無法判斷方向與停留時間，也不易擴展至大範圍場域。
- **深度學習影像辨識的成熟：** 現今物件偵測技術（如 YOLO v8）與時間序列預測技術（如 GRU、VMD-GRU）可應用於即時場域分析與預測。

---

---

## 4. 問題與解決方法

| 問題 | 解決方法 |
|------|-----------|
| 影像中人物辨識準確率低 | 使用 YOLOv8 進行即時人物偵測並優化模型參數 |
| 多人重疊、遮蔽造成追蹤困難 | 使用 Deep SORT 進行多目標追蹤 |
| 人流數據波動大、難預測 | 將數據透過 VMD 分解為穩定子序列，再用 GRU 預測 |
| 難以視覺化與展示結果 | 建立 Web Dashboard 或本地圖形介面展示即時人流與預測結果 |

---

## 5. 實現步驟、方法、技術

**系統架構：**  
影像擷取 → 人物偵測（YOLOv8）→ 人物追蹤（Deep SORT）→ 資料記錄 → 預測模型（VMD + GRU）→ 資料可視化與應用

**主要模組與技術：**

| 功能 | 技術 / 工具 |
|------|--------------|
| 影像擷取 | CCTV / IP Camera |
| 影像辨識 | YOLOv8 |
| 目標追蹤 | Deep SORT / ByteTrack |
| 資料記錄與整理 | Pandas / SQLite |
| 預測模型 | GRU / VMD-GRU |
| 可視化與介面 | Matplotlib / Plotly / Dash / Streamlit |
| 環境佈署 | Python + Flask / Raspberry Pi |

---

## 6. 預期結果

- 可即時顯示場域中人數、人流方向與滯留時間。  
- 提供人流趨勢預測圖，支援短期（ 10~30 分鐘）人數預估。  
- 系統可部署於筆記型電腦或邊緣裝置（ Raspberry Pi）上進行展示（Demo）
